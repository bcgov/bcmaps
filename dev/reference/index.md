## Utility functions

Various utility functions for working with BC data

- [`bc_area()`](http://bcgov.github.io/bcmaps/dev/reference/bc_area.md)
  : The size of British Columbia
- [`bc_bbox()`](http://bcgov.github.io/bcmaps/dev/reference/bc_bbox.md)
  : Get an extent/bounding box for British Columbia
- [`transform_bc_albers()`](http://bcgov.github.io/bcmaps/dev/reference/transform_bc_albers.md)
  : Transform a Spatial\* object to BC Albers projection
- [`utm_convert()`](http://bcgov.github.io/bcmaps/dev/reference/utm_convert.md)
  : Convert a data.frame of UTM coordinates to an sf object with a
  single CRS
- [`combine_nr_rd()`](http://bcgov.github.io/bcmaps/dev/reference/combine_nr_rd.md)
  : Combine Northern Rockies Regional Municipality with Regional
  Districts
- [`bc_neighbours()`](http://bcgov.github.io/bcmaps/dev/reference/bc_neighbours.md)
  : Boundary of British Columbia, provinces/states and the portion of
  the Pacific Ocean that borders British Columbia
- [`bec_colours()`](http://bcgov.github.io/bcmaps/dev/reference/bec_colours.md)
  [`bec_colors()`](http://bcgov.github.io/bcmaps/dev/reference/bec_colours.md)
  : Biogeoclimatic Zone Colours

## Raster utilities

Functions for specific raster operations

- [`raster_by_poly()`](http://bcgov.github.io/bcmaps/dev/reference/raster_by_poly.md)
  : Overlay a SpatialPolygonsDataFrame or sf polygons layer on a raster
  layer and clip the raster to each polygon. Optionally done in parallel
- [`summarize_raster_list()`](http://bcgov.github.io/bcmaps/dev/reference/summarize_raster_list.md)
  : Summarize a list of rasters into a list of numeric vectors
