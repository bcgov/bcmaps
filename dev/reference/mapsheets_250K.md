# NTS 250K Grid - Digital Baseline Mapping at 1:250,000 (NTS)

## Usage

``` r
mapsheets_250K()
```

## Source

https://open.canada.ca/data/en/dataset/055919c2-101e-4329-bfd7-1d0c333c0e62

## Value

The spatial layer of `mapsheets_250K` as an `sf` object.
